A web-based application that encrypts and decrypts images using **pixel-level bit manipulation techniques**. The tool allows users to securely upload an image, apply encryption, preview the encrypted output, and decrypt it back to the original image.

## Features
- Image encryption using pixel-level bit manipulation
- Reversible decryption to restore original images
- Upload, preview, encrypt/decrypt, and download functionality
- Simple and user-friendly web interface
